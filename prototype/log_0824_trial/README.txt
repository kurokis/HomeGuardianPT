
# 車両構成
1/18ラジコンの右側にToF(VL53L0X)と超音波センサ(HC-SR04)を一つずつ搭載
モータは動かさず、手で押して計測

# プログラム
Arduino互換ボードにlogger_0824.inoを書き込んで測定

# データフォーマット
 - 1列目: マイコン起動からの経過時間（単位: ミリ秒）
 - 2列目: ToFで読み取った距離（単位: ミリメートル）
 - 3列目: ToFで読み取った距離（単位: ミリメートル）

# ログデータ
 - DATALOG_1lap_right_original.csv: 右手法のルートでコース一周させたときのログ
 - DATALOG_1lap_right_trimmed.csv: 上記のタイムスタンプが何度かリセットされていたので、綺麗な部分だけを抜き出したログ
 - DATALOG_1lap_shortest.csv: 今考えている最速ルートでコース一周させたときのログ
 - DATALOG_straight.csv: スタート地点から壁と平行に直進しつづけたときのログ
